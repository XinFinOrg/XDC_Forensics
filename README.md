## Dev setup
UI of XDC Forensics monitoring

```
nvm use
npm run install:clean
npm run dev
```

Go to http://localhost:8080

## Host on server
1. Install serve
```
npm install -g serve
```

2. Run the build
```
npm run start
```
It shall now runs on port 8888
